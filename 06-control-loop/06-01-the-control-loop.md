# The Control Loop

This chapter states the control loop as a reference model: continuous comparison of system state against intent using evidence, with explicit decision points. Diagrams in `diagrams/` illustrate placeholders for this model.

## Why this matters

A reference model aligns teams on what “continuous certification” means operationally.
