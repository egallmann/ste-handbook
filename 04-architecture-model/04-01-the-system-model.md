# The System Model

This chapter describes the system model carried by Architecture IR: entities, identities, and the holistic graph that represents the architecture under discussion. It focuses on conceptual structure, not a specific serialization.

## Why this matters

The system model is the referent for traceability, diff, and conformance. Readers must understand its role separately from any single projection.
