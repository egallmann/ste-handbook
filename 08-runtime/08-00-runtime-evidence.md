# Runtime Evidence

This chapter describes runtime evidence types, collection assumptions, and how evidence binds to IR elements and traces. It avoids vendor-specific stacks unless used as examples.

## Why this matters

Conformance without evidence is hollow. This chapter grounds assessment in observable signals.
