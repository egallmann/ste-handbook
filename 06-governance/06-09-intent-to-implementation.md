# Intent to Implementation

This chapter traces how intent compiles and constrains implementation work: contracts, scaffolding, and enforcement points. It bridges intent, IR, and engineering execution without prescribing a specific toolchain.

## Why this matters

The handoff from intent to code is where many systems lose traceability. This chapter names the interfaces.
