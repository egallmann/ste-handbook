# STE Handbook

The STE Handbook is the primary technical documentation for the
System of Thought Engineering (STE).

System of Thought Engineering (STE) is a model-driven architecture and
governance system that encodes architectural intent as machine-readable
decisions, compiles those decisions into a canonical architecture model,
enforces constraints and invariants at runtime, and validates system
behavior against architectural intent using deterministic evidence and
conformance testing.

This repository explains the concepts, architecture, lifecycle,
and usage of STE. The normative specification for STE is defined
in the ste-spec repository.
